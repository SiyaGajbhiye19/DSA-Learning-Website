Fibonacci Search

            int min(int x, int y) { return (x <= y) ? x : y; }
            
            int fibMonaccianSearch(int arr[], int x, int n) {
                int fibMMm2 = 0; 
                int fibMMm1 = 1; 
                int fibM = fibMMm2 + fibMMm1; 
            
                while (fibM < n) {
                    fibMMm2 = fibMMm1;
                    fibMMm1 = fibM;
                    fibM = fibMMm2 + fibMMm1;
                }
            
                int offset = -1;
            
                while (fibM > 1) {
                    int i = min(offset + fibMMm2, n - 1);
            
                    if (arr[i] < x) {
                        fibM = fibMMm1;
                        fibMMm1 = fibMMm2;
                        fibMMm2 = fibM - fibMMm1;
                        offset = i;
                    }
                    else if (arr[i] > x) {
                        fibM = fibMMm2;
                        fibMMm1 = fibMMm1 - fibMMm2;
                        fibMMm2 = fibM - fibMMm1;
                    }
                    else
                        return i; 
                }
            
                if (fibMMm1 && arr[offset + 1] == x)
                    return offset + 1;
            
                return -1; 
            }

Description

Fibonacci Search is an efficient searching algorithm that works on sorted arrays. It utilizes Fibonacci numbers to divide the search space, making it similar to Binary Search but with a more dynamic approach.

Steps:

  • Find the smallest Fibonacci number greater than or equal to n (size of the array).

Initialize search variables:

  • fibMMm2 = 0 (Fibonacci m-2)
  • fibMMm1 = 1 (Fibonacci m-1)
  • fibM = fibMMm2 + fibMMm1 (Fibonacci m)

Search the element using Fibonacci numbers:

  • Compare arr[i] with x.
  • If arr[i] < x, shift the search range to the right.
  • If arr[i] > x, shift the search range to the left
  • If arr[i] == x, return the index.
  • Check the last element if needed.
  • Return the index of x if found, otherwise return -1

Time Complexity:

  • Best Case: O(n) , Average Case: O(log N) , Worst Case: O(log N)

Example

Input:

arr[] = {5 ,10 , 15 ,20 , 25 , 30 , 35}
            Enter the element to search: 20

Step-by-Step Process of Fibonacci Search:

Find the smallest Fibonacci number ≥ 7 (size of array)

  • Fibonacci numbers: 0, 1, 1, 2, 3, 5, 8, 13...
  • Smallest Fibonacci number ≥ 7 is 8 → fibM = 8.
  • Set offset = -1 and start searching using Fibonacci numbers.

Compare arr[i] with x = 20:

  • Step 1: i = min(2, 6) = 2, arr[2] = 15 → Move right
  • Step 2: i = min(3, 6) = 3, arr[3] = 20 → Element found

Output:

Found at index: 3
