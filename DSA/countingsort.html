Counting Sort

Code

        
        #include <stdio.h> 
        #include <stdlib.h>
    
            void countSort(int inputArray[], int N) {
                
                int M = 0;
                for (int i = 0; i < N; i++)
                    if (inputArray[i] > M)
                        M = inputArray[i];
                int* countArray = (int*)calloc(M + 1, sizeof(int));
                for (int i = 0; i < N; i++)
                    countArray[inputArray[i]]++;
                for (int i = 1; i <= M; i++)
                    countArray[i] += countArray[i - 1];
            
                int* outputArray = (int*)malloc(N * sizeof(int));
                for (int i = N - 1; i >= 0; i--) {
                    outputArray[countArray[inputArray[i]] - 1] = inputArray[i];
                    countArray[inputArray[i]]--;
                }
            
                for (int i = 0; i < N; i++)
                    inputArray[i] = outputArray[i];
            
                free(countArray);
                free(outputArray);
            }
            int main() {
                int N;
            
                printf("Enter the number of elements in the array: ");
                scanf("%d", &N);
            
                int inputArray[N];
            
                printf("Enter the elements of the array:\n");
                for (int i = 0; i < N; i++) {
                    scanf("%d", &inputArray[i]);
                }
                countSort(inputArray, N);
                printf("Sorted array is:\n");
                for (int i = 0; i < N; i++)
                    printf("%d ", inputArray[i]);
                
                return 0;
            }

Description

Counting Sort is a non-comparative sorting algorithm that sorts an array by counting the occurrences of each element and using this information to place them in the correct order.

Steps:

  • Find Maximum Element – Determine the largest value in the array.
  • Create Count Array – Initialize an array to store the frequency of each element.
  • Compute Prefix Sum – Modify the count array to store the cumulative sum, which helps in determining the correct position of each element.
  • Sort Elements – Traverse the original array from right to left, placing each element in its correct position using the count array.
  • Copy Sorted Elements – Transfer sorted elements back to the original array.
  • Time Complixity is O(N+M)

Example

Input:

arr[] = {4 3 12 1 5 5 3 9}

Step-by-Step Process of Counting Sort:

Process:

  • Find Maximum Element: max = 12
  • Create Count Array: [0,1,0,2,1,2,0,0,0,1,0,0,1]
  • Compute Prefix Sum: [0,1,1,3,4,6,6,6,6,7,7,7,8]
  • Sort Elements: [1, 3, 3, 4, 5, 5, 9, 12]
  • Copy Sorted Elements Back to Original Array

Output:

Sorted array is:
            1 3 3 4 5 5 9 12

This is the sorted array in ascending order.
